Les boucles

Les instructions for et while qui sont deux types de boucles les plus utilisées en js, servent à repeter plusieurs lignes de code

La boucle for

For pre,ds 3 paramètres encadrés dans des parenthèses, suivi entre accolades des instructions à executer autant de fois que la boucle l'indique. les paramètres sont:

  • Initialisation: ici on declare la variable qui servira de compteur
  • Condition: Tant que cette condition est vraie la boucle s'executera
  • Expression finale: expression qui est éxecutée à chaque fin de boucle
for(initialisation; condition; incrémentation){ instruction 1; instruction 2; }
