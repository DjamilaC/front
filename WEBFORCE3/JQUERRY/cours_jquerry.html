Selectionner en jquerry

le coeur de jQuerry se trouve dans la selection d'elements pour ensuite executer des méthodes dessus

$(selecteur) accepte en argument tout sélecteur CSS, jQuerry nous renvoie ensuite une collection d'un ou plusieurs éléments DOM enveloppées dans un objet jQuerry.

$('div') séléctionne tous les divs $('.titre') séléctionne tous les éléments qui ont la classe="titre". $('#boite') séléctionne l'unique élement qui a l'id="boite"

cette enveloppe jQuerry est celle qui donne accés à toutes les fonctionnalités de jQuerry

  • Rajouter des classes
  • Modifier le contenu HTML
  • Afficher ou cacher des élements
  • les animer etc.

Tous les elements sélectionnées en javascript pur (document.getElementById, querrySelector etc) sont des élements DOM et tous ceux séléctionnés avec $('...') sont des éléments dans une enveloppejQuerry

jQuerry dispose de plusieurs méthodes pour parcourir l'arbre du document (le DOM)

  • children() parcourt les enfants
  • find() parcourt les descendants
  • next() renvoie le voisin suivant
  • prev() renvoie le voisin précédent
  • parent() renvoie le parent
  • eq() renvoie, parmi une liste d'éléments, celui qui se trouve à l'index fourni
