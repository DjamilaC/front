Les instructions

un script js est un fichier contenant des instructions placées les unes à la suite des autres. chacune de ces instructions peut etre vue comme un ordre donné au moteur javascript, une action que vous souhaitez réaliser, à la manière d'une recette de cuisine

  • faire fondre le beurre
  • Mélanger le beurre et la farine
  • Verser le chocolat petit à petit etc...

les instructions se suivent et sont effectuées les à la suite des autres, Du haut vers le bas(comme en css), le browser parse (analyse) dans l'ordre du flux.

chaque instruction DOIT se terminer par un point virgule; afin d'indiquer au moteur javascript que cette instruction est terminée

Les fonctions

Les fonctions permettent de regrouper une série d'instructions pour pouvoir les réutiliser. quand on a besoin du code d'une serie d'instructions, plutot que de réecrire à chaque fois toutes les instructions, on appellera une fonction par son nom (faireUnGateau. cette fonction contient l'ensemble des instructions à executer

par exemple la fonction alert est une fonction native ( intégrée au navigateur) qui contient en gros les instructions suivantes

  • Creer le cadre de la boite de dialogue
  • Positionner le cadre dans la fenetre
  • placer un bouton avec le texte ok et un evenement click
  • créer une zone de texte qui affichera le texte de l'art à présenter au visiteur

Nous on a juste besoin d'utiliser le fonction alert() en lui donnant un paramètre exemple alert("coucou"). la foction executera toutes les instructions à chaque fois.
on peut utiliser cette fonction alert quand on veut en la declanchant avec alert(quelque chose). ce (quelque chose) est un ARGUMENT attendu par la fonction.

var maVariable = document.querySelector('h2'); alert('coucou');

Les fonctions querrySelector et alert sont disponibles dans le navigateur, ce sont des methodes

Si vous voyez quelque chose qui ressemble à une variable et qui est suivie de parenthèses c'est une fonction.

les fonctions prennent souvent des arguments- des bous de données dont elles ont besoin pour faire leur travail. les arguments sont placés entre parenthèses, séparés par des virgules s'il yen a plusieurs.

Par exemple la fonction alert() fait apparaitre une poup mais vous devez donner une chaine comme argument pour indiquer à la fonction ce que vous souhaitez ecrire dans la fenêtre

Nous pouvons écrire nos propres fonctions
